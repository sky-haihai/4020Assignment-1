Environment: Windows 7/8/10/11, Python 3.11.0, Vitual Studio Code (Optional)

Option 1: VS Code
0. Open Project Folder inside VS Code
1. Install the basic Python extension
2. Run "Python: Select Iterpreter" command via VS Code (default hot key is F1)
3. Open a new terminal and Type "py main.py" to execute the script
4. Once finished, the result will be saved to the path: "res/group1_result.xml" relative to the root directory

Option 2: Windows Power Shell

1. Change directory to the root directory of the project
2. Execute "py main.py"
3. Once finished, the result will be saved to the path: "res/group1_result.xml" relative to the root directory
